# 10 SIMPLE START GUIDE

## 1) Login

1. Open `http://127.0.0.1:18080`.
2. Enter username and password.
3. Click **Sign in**.

## 2) See LAB framework

1. Go to **Admin → Frameworks**.
2. Confirm only **LAB** is listed.

## 3) Create first project from LAB

1. Go to **Projects**.
2. Click **Create project**.
3. Select framework **LAB**.
4. Keep initialize option enabled and submit.

## 4) Open project worklist

1. Open your project.
2. Click **Go to worklist**.

## 5) Open indicator

1. In worklist, click any indicator card.
2. Indicator details open.

## 6) Add evidence

1. Open **Evidence** section.
2. Click **Add evidence**.
3. Fill details and save.

## 7) Request review

1. Open **Actions** section.
2. Click **Send for Review**.

## 8) Approve/finalize (authorized roles only)

1. Reviewer/approver checks evidence.
2. Use **Mark as Met** when readiness is complete.

## 9) Check readiness

1. Go to project **Readiness** page.
2. Check score and risk blocks.

## 10) Use exports/inspection pages

1. Open **Export history** for export jobs.
2. Open **Inspection mode** for MET-only inspection view.

## 11) If a page shows an error

1. Refresh once.
2. Confirm you are still on `http://127.0.0.1:18080`.
3. Sign out and sign in again.
4. If error stays, capture screenshot and report route + action.
