# Project name options

## Recommended
### 1. AccrediOps
Best fit for internal operations and service delivery.

## Strong alternatives
### 2. AccrediDesk
More workbench-like and operational.

### 3. AccrediFlow
Good if workflow language is preferred.

### 4. EvidenceOps
Good if you want to emphasize evidence-centered operations.

### 5. ComplianceWorkbench
Descriptive, but longer and less brandable.

### 6. AccrediForge
Stronger brand feel, less literal.

## Recommendation
Use:

- product working name: **AccrediOps**
- repo name: **accrediops**
- future internal project label: **AccrediOps Core**
