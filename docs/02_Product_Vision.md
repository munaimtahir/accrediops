# Product vision

## High-level vision

Build an internal accreditation operations platform that helps your team prepare, manage, and deliver client accreditation work efficiently, with strong indicator-level control, evidence discipline, recurring documentation tracking, reuse across clients, and print-ready inspection output.

## How the product should feel

The system should behave less like a normal dashboard and more like an **accreditation workbench**.

A user should be able to:
- load a framework
- see all indicators
- understand what type of evidence each indicator needs
- know which indicators are one-time versus recurring
- upload, link, generate, or track evidence
- monitor which items are complete and which are under process
- prepare the final hard-evidence pack in order
- clone/reuse the same framework for another client

## What the product is not

The product is not:
- a client portal
- a SaaS offering
- a general DMS
- a generic task board
- an AI autopilot
- a paperless-only compliance fantasy

## Strategic differentiator

The strongest differentiator is reusable accreditation intelligence:
once one framework is fully structured and completed for one client, the same framework and its evidence/document patterns can be reused for future clients with controlled client-specific replacement.
