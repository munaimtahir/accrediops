Internal Accreditation Operations

AccrediOps is live on the former OPS slot.

This deployment currently exposes the working Django backend on the reused OPS API port and a lightweight operator shell on the reused OPS frontend port. The full product frontend is still to be built inside this repository.

What is available now

  • Django admin for internal data entry and review.
  • REST API endpoints under /api/.
  • SQLite-backed starter environment with existing migrations applied.
  • Stable local entrypoint on 127.0.0.1:18080 through Caddy.
